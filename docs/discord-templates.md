# Discord — Community Outreach Templates

Use these in AI agent builder Discords (LangChain, CrewAI, AutoGen, OpenClaw, Claude Code, etc.). Don't lead with a link. Be a real member first; share when relevant.

---

## When to post

- In **#showcase**, **#projects**, **#built-with-[framework]** channels, if they exist.
- After you've been active in the server (answered questions, shared useful takes).
- One server per day max. Never spam multiple servers in one day.

---

## Rules

- **Read the server rules first.** Some disallow self-promo or limit it to specific channels.
- **Don't @everyone or @here.** Ever.
- **One message, one link.** Don't repeat the link in follow-ups unless someone asks.
- **Respond to every reply.** Be helpful, not salesy.
- If your post is removed, DM a mod politely and ask what would be allowed (e.g. a specific channel or day).
